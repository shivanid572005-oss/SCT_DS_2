# SCT_DS_2
## Project Overview
This project is part of the SkillCraft Technology Data Science Internship.

The objective of this task is to perform Data Cleaning and Exploratory Data Analysis (EDA) on the Titanic Dataset and identify patterns and trends in the data.

## Dataset
Titanic Dataset

## Tools and Libraries
- Python
- Pandas
- Matplotlib
- Seaborn

## Data Cleaning
- Checked for missing values
- Filled missing Age values using median
- Filled missing Embarked values using mode
- Replaced missing Cabin values with "Unknown"
- Removed duplicate records

## Exploratory Data Analysis
The following analyses were performed:

1. Survival Count Analysis
2. Gender Distribution Analysis
3. Survival by Gender
4. Passenger Class Distribution
5. Survival by Passenger Class
6. Age Distribution
7. Fare Distribution
8. Correlation Heatmap

## Key Findings
- Female passengers had a higher survival rate than males.
- First-class passengers had better survival chances.
- Most passengers were between 20 and 40 years old.
- Higher fares were associated with higher survival rates.
- Passenger class significantly affected survival.
